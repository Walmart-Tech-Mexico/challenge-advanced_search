# Busqueda avanzada

**Nivel:** Intermedio

## Descripción
Buscar productos con múltiples filtros: categoría, precio, marca.

## Objetivo
Implementa una solución en Java que cumpla con la lógica descrita. Usa la plantilla `Main.java` para comenzar.

## Cómo empezar
1. Clona este repositorio.
2. Dirígete a la carpeta `Busqueda_avanzada`.
3. Abre `Main.java` y escribe tu solución.
4. ¡Comparte tu solución con la comunidad!
